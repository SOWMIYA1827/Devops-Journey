# 🐧 Linux Basic Commands

## Introduction

Linux is an open-source operating system widely used in servers, cloud computing, and DevOps. These are the basic commands every beginner should know.

In this guide, I practiced Linux commands using **Windows Subsystem for Linux (WSL)** on **Windows 11**, which allows Linux to run directly on Windows without installing a separate Linux operating system.

---

# 💻 Learning Environment

- **Operating System:** Windows 11
- **Linux Environment:** Ubuntu using Windows Subsystem for Linux (WSL)
- **Editor:** Visual Studio Code
- **Version Control:** Git
- **Repository Hosting:** GitHub



## 1. pwd

Print the current working directory.

```bash
pwd
```

Example Output:

```
/home/sowmiya
```

---

## 2. ls

List files and folders.

```bash
ls
```

Options:

```bash
ls -l
ls -a
ls -la
```

---

## 3. cd

Change directory.

```bash
cd Documents
```

Go back:

```bash
cd ..
```

Go to home directory:

```bash
cd ~
```

---

## 4. mkdir

Create a new directory.

```bash
mkdir DevOps
```

---

## 5. rmdir

Remove an empty directory.

```bash
rmdir DevOps
```

---

## 6. touch

Create a new file.

```bash
touch notes.txt
```

---

## 7. cat

Display file contents.

```bash
cat notes.txt
```

---

## 8. cp

Copy files.

```bash
cp file1.txt file2.txt
```

---

## 9. mv

Move or rename files.

```bash
mv old.txt new.txt
```

---

## 10. rm

Delete files.

```bash
rm notes.txt
```

Delete folder recursively:

```bash
rm -r FolderName
```

---

## 11. clear

Clear terminal.

```bash
clear
```

---

## 12. history

View command history.

```bash
history
```
