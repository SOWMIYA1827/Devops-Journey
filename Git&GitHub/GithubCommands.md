## 🌿 Branch Management

- Create Branch

```bash
git branch feature-login
```

- Switch Branch

```bash
git switch feature-login
```

- Create & Switch Together

```bash
git switch -c feature-login
```

- List Branches

```bash
git branch
```

- Delete Branch

```bash
git branch -d feature-login
```

## 📜 Git Log

View commit history.

```bash
git log
```

One line format

```bash
git log --oneline
```

Graph View

```bash
git log --oneline --graph --all
```

## 🏷 Git Tags

Create Version Tags

Create tag

```bash
git tag v1.0.0
```

List tags

```bash
git tag
```

Push tag

```bash
git push origin v1.0.0
```

Push all tags

```bash
git push origin --tags
```
